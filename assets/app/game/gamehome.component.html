Games

In these pages you can find some of the games I have made, along with explanations of the design of each game and some design decisions that I made. For ongoing projects there are also dev blogs to show progress and updates. If possible a copy of the game will be included (or the latest playable build).

Eruption

Eruption

Eruption is a frantic, fast paced puzzle game. The aim is to protect as many cities as possible from the approaching lava by diverting it through trenches.

Migration

Migration - Fish level 1

Migration is a relatively casual game (the difficulty needs to be tweaked to make it more so), in which you guide 3 animals through their migrations, flying swimming and running to their destinations while avoiding predators and battling the elements.

Survivor

Survivor - First area

After crash landing on a mysterious group of seemingly uninhabited islands, you must find a way to get home. Explore the islands and try to survive long enough to escape!
